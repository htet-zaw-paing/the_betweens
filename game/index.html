Operational Directives

I. OPERATIONAL SCOPE

"The Betweens" is structured as a self-contained narrative simulation. Core runtime operations execute dynamically through offline protocol functionality. Minimal online connectivity is established solely for fetching dynamic external assets. The primary architecture ensures local, client-side rendering execution.

II. PRIVACY & DATA TELEMETRY

The system operates strictly on a zero-knowledge framework regarding user telemetry. We do not harvest, track, transmit, or monetize personal identification data. Local hardware interactions (e.g., Camera access for rendering screenshots, Audio interface for playback) remain confined entirely within the client-side environment. At no point is optical, acoustic, or gameplay data uploaded to our remote servers.
